AES-Crypto
==========
Java application used to encrypt and decrypt files using AES with 128-bit keys. 

The input key file must be one line and must have 32 hex characters. The input file can be multiple lines and can have up to 32 characters where all are hex characters, if a line has more than 32 characters it is skipped. When you run the program the current state after each step is printed out as a 2d matrix.
